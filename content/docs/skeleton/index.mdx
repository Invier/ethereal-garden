### When To Use

Use the Skeleton component when you need to display a loading state for a component.
- Display a loading state for a component
- Display a loading state for a list of items
- Display a loading state for a table
- Display a loading state for a chart
- Display a loading state for a modal
- Display a loading state for a form
- Display a loading state for a table
